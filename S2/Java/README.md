# S2 Sae Dev App Ab7

## Installation

1. Cloner le projet 
```
$ git clone https://git.iut-orsay.fr/psounou/s2-sae-dev-app-ab7.git
$ cd s2-sae-dev-app-ab7 
```

2. Ouverture </br>
Vous pouvez ouvrir le prjet avec eclipse ou un autre IDE java (comme Intellij IDEA)

Avec Eclipse : </br>
> File > Open Projects from File

3. Tests </br>

⚠️  Il faut suivre la configuration suivante 
>1. Dans Project, assurez-vous que l’option Build automatically est activée.</br>
>2. Dans File > Properties, cliquez sur Java Build Path.</br>
>3. Dans l’onglet Source, cliquez sur le bouton Add Folder et ajoutez le >répertoire tests/.</br>
>4. Nous avons besoin des assertions pour les tests unitaires mais assert est ignoré par défaut en Java, il faut donc ajouter l’option -ea (enable assertions) dans Run > Run configurations... > Arguments > VM arguments.
