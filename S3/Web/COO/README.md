## Indications Techniques

### ⚠️ Avant le lancement de l'application ⚠
  
  * Assurer vous d'avoir ajouter le fichier **gson-2.10.1.jar** au dépendance du projet **après avoir ouvert le projet Java** (Fichier gson-2.10.1.jar fourni dans le rendu)

    - Etapes à suivre dans votre application **intelliJ**: 
      * suivez ce chemin : 
        - file 
        - project structure 
        - modules 
        - dependencies 
        - + 
        - jar or directories 
        * selectionnez le fichier gson-2.10.1.jar
        - apply 
        - ok
        * vous pouvez désormais lancer le projet

    - Etapes à suivre dans votre application **Eclipse**: 
      * suivez ce chemin : 
        * clic droit sur le projet dans l'explorateur de gauche
        - Build Path
        - Configure BuildPath
        - Libraries
        - Add External JARs 
        * selectionnez le fichier gson-2.10.1.jar
        - Apply and Close
        * vous pouvez désormais lancer le projet 
  
## Indications de compréhension

    * Le fichier gson-2.10.1.jar est la bibliothèque Google Gson, elle nous aide à utilsé le json plus simplement
